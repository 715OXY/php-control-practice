# php-basics-practice

## 概要
COACHTECH 教材 Tutorial 7-2「制御構文 ハンズオン演習」で作成した成果物です。
配列型で与えられた学生の成績に応じて、評価（５段階）と合否を判定するプログラムです。
個別成績（名前、点数、評価、判定）と統計情報（合格者・不合格者数、平均点）をWeb上に表示します。

## 使用技術
- PHP 8.2
